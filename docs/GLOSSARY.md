# CamTraffic Glossary

Task: 379
Last Updated: 2026-07-10

| Term | Definition |
|---|---|
| AI Service | FastAPI service that runs detection and OCR pipelines. |
| ANPR | Automatic Number Plate Recognition for license-plate text extraction. |
| Appeal | Driver-submitted dispute against an issued violation/fine. |
| Camera Health | Operational status of a camera endpoint (online/offline/degraded). |
| CER | Character Error Rate metric for OCR quality evaluation. |
| Detection | AI-identified traffic sign/vehicle event from an image/frame. |
| Driver Portal | End-user interface for drivers to view fines and submit appeals. |
| Evidence | Image/metadata attached to detections and violations. |
| Fine | Monetary penalty generated from an approved violation. |
| mAP@50 | Mean Average Precision at IoU 0.50 for detection evaluation. |
| Mock Mode | Development mode where AI returns deterministic sample outputs. |
| OCR | Optical Character Recognition used for plate text extraction. |
| Officer Portal | Interface for officers to review detections and violations. |
| RBAC | Role-Based Access Control authorization model. |
| SSE | Server-Sent Events used for live feed updates in frontend views. |
| UAT | User Acceptance Testing with role-based workflow validation. |
| Violation | Confirmed traffic rule infringement linked to evidence. |
| VPS | Virtual Private Server used for production deployment. |
