## 2. Core Workflows

1. Login and open driver dashboard.
2. View violations and evidence.
3. View and pay fines.
4. Submit an appeal with supporting note.
5. Track appeal decision status.
6. Review notifications and vehicle history.

## 4. Troubleshooting

- If violations are missing, confirm vehicle is linked to your account.
- If payment history does not refresh, reload page and verify backend health.
- If appeal submission fails, verify violation status is appeal-eligible.
