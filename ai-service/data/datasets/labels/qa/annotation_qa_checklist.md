# Annotation QA Checklist (Task 130)

Use this checklist for each annotation batch.

- [ ] Class IDs match `labels/yolo/classes.txt`
- [ ] Bounding boxes are normalized to YOLO format
- [ ] No negative or >1 coordinates
- [ ] No duplicate boxes for the same object
- [ ] At least 10% sample double-reviewed
- [ ] OCR plate boxes include corresponding transcription metadata
- [ ] Difficult samples tagged (`blurred`, `occluded`, etc.)
- [ ] Batch disagreement rate recorded
