# Annotation Labels (Task 130)

This folder stores annotation assets for detection and OCR preparation.

## Structure

```text
labels/
├── yolo/
│   ├── classes.txt
│   ├── class-map.json
│   └── examples/
│       └── sample_000001.txt
├── cvat/
│   ├── README.md
│   └── project-labels.json
└── ocr/
    └── README.md
```

## Notes

- Final per-image label files should be generated by your annotation platform export.
- Keep `classes.txt` stable once training starts to avoid class-ID drift.
