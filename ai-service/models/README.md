# Place trained YOLOv11 weights in this directory.

Expected default filename: `yolov11_camtraffic_v1.pt`

Configure with environment variables:

- `AI_MODEL_PATH` — directory containing weights (default: `models`)
- `AI_YOLO_WEIGHTS` — weights filename or absolute path

The backend seed data references `models/yolov11_traffic_signs_v1.pt` for the active
`YOLOv11 Traffic Signs v1.0.0` model version.
