# @camtraffic/utils

Shared utility functions for CamTraffic frontends.

## Exports

- `cn()` — class name helper
- `formatDate`, `formatDateTime`, `formatRelativeTime`
- `formatCurrency`, `formatPlateNumber`, `formatFileSize`
- `toIntlLocale`, `isAppLocale`
- `slugify`
- `isValidEmail`, `isRequired`, `minLength`
- `isValidKhmerPhone`, `normalizeKhmerPhone`

```ts
import { cn, formatFileSize, slugify } from '@camtraffic/utils';
```
